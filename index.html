FEATURES

Uploading Exam Results to a Database

EMS provides a user friendly interface where teachers can upload .csv files.
This files are processed in the background and its contents are saved to the
results table of the EMS database

Create Customized Dashboards

EMS provides the functionality to create customized dashboards. Interactive
graphs or charts can be added to the dashboard to provide insights into student's
data.
